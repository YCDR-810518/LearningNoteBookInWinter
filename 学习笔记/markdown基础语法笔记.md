# markdown基础语法

## 1. 标题
#开头+空格+内容，表示标题（#的数量表示标题层级（最多6级标题））

## 2. 文本格式
* **加粗**：用两个星号（ * ）包裹
* *斜体*：用一个星号包裹
* ~~删除线~~: 用两个波浪号（ ~ ）包裹会给文本带上删除线

## 3. 列表
* **无序列表**：使用 * 加上空格
	* A
	* B
* **有序列表**：使用数字 + . + 空格
	1. 第一个
	2. 第二个

## 5. 代码段&代码块

* **行内的代码**：`print('hello python!')`用两个反引号包裹住
* **代码块**：三个反引号包裹，并在开头的三个反引号后写明代码块的语言
	**C语言的代码块**
	``` c
	include <stdio.h>
	printf("Hello world!");
	return 0;
	```
	**python的代码块**
	
	``` python
	print('Hello pyhon')
	```
	
## 6.引用&分割线
* >**引用**：一个引用块，用于摘抄&引用（使用** > **符号放在要引用的句子之前）
* **分割线**：三个/更多的短横线
